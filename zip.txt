# Build on your Mac is for GitHub push only — do NOT zip node_modules from Mac to Linux.
# Use GitHub Actions artifact instead (Linux binaries).

#
# 2. Upload to Hostinger (from your Mac):
#    scp chms-deploy.tar.gz u421975600@YOUR_HOSTINGER_IP:~/domains/api.paggglobal.org/
#
# 3. On Hostinger SSH:
#    cd ~/domains/api.paggglobal.org
#    rm -rf public_html/*
#    tar -xzf chms-deploy.tar.gz -C public_html
#    nano public_html/.env
#    cd public_html && ./node_modules/.bin/prisma migrate deploy
#    ./node_modules/.bin/tsx prisma/seed.ts   # first time only
#
# 4. hPanel Node.js apps:
#    api.paggglobal.org → apps/api → dist/main.js
#    paggglobal.org → apps/web/.next/standalone/apps/web → server.js
